Para poder usar la API, debemos instalar la extensión en Visual Studio Code llamada: Live Server, de RitWick Dey
Seguidamente tenmos que abrir el archivo (Open File) y abrir el archivo con el HTML, css y el JS. Después, abrir el HTML y hacer click derecho
en el editor y darle a la opción Open With Live Server. Se nos conectara de forma local a la api.
Abrimos la pagina en el navegador y listo para usar.
